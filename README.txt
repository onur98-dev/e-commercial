 E-Commerce Website (React)

A modern and responsive e-commerce website built with **React**. This application features a clean UI, live product fetching from a mock API, cart functionality, and smooth navigation.

---Features:
-  Built with React and functional components
-  Fully responsive design for all screen sizes
-  Live data fetching from an external API
-  Product listing and detail pages
-  Add to cart, remove from cart with real-time counter
-  Light/Dark mode toggle support
-  Seamless navigation with React Router
-  State management using Redux
-  Drawer-based shopping cart UI

Technologies Used:

- **React** (with Hooks)
- **React Router DOM**
- **Redux Toolkit**
- **Material UI** (for drawer and badges)
- **React Icons**
- **CSS Modules**
- **JavaScript (ES6+)**
- **API
